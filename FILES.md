## 🎯 Какой файл использовать?

### Для начала работы
1. Прочитайте `QUICKSTART.md`
2. Настройте `.env` по примеру `.env.example`
3. Запустите `quick_test.py`

### Для разработки
1. Изучите `chat_bot.py` (ООП подход)
2. Или `test_chat.py` (функциональный подход)
3. Смотрите примеры в `PROJECT_STRUCTURE.md`

### Для тестирования
1. `quick_test.py` - быстрая проверка
2. `test_persistence.py` - полный тест
3. `TEST_ALL.bat` - запуск всех тестов

### Для демонстрации
1. `demo_persistence.py` - интерактивная демонстрация
2. Запустите несколько раз подряд

### Для интеграции
1. Импортируйте `ChatBot` из `chat_bot.py`
2. Или используйте функции из `test_chat.py`
3. Примеры в `FEATURES.md`
