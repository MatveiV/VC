# История изменений

## Версия 2.3 - Выбор моделей и работа с файлами (2026-03-01)

### ✨ Новые возможности

#### 1. Выбор моделей ИИ
- **Поддержка 9 моделей**: 4 OpenAI + 5 Claude
- **Автоматическое определение** возможностей модели
- **Динамическая смена модели** во время работы
- **Информация о моделях**: стоимость, контекст, возможности

#### 2. Работа с файлами
- **Текстовые файлы**: 30+ форматов (py, js, md, json, и др.)
- **Изображения**: jpg, png, gif, webp (для моделей с vision)
- **Автоматическое определение** типа файла
- **Анализ и преобразование** файлов моделью ИИ

### 📝 Новые файлы

- `models_config.py` - конфигурация всех доступных моделей
- `file_handler.py` - модуль для работы с файлами
- `demo_models_and_files.py` - интерактивная демонстрация
- `test_models_and_files.py` - тесты новой функциональности

### 🔧 Изменения в API

#### chat_bot.py
- Новый параметр `file_path` в методе `chat()`
- Новые методы:
  - `set_model(model_id)` - смена модели
  - `get_current_model()` - получить текущую модель
  - `list_available_models()` - список моделей
- Поддержка vision API для изображений

#### Доступные модели

**OpenAI:**
- gpt-4o (с vision)
- gpt-4-turbo (с vision)
- gpt-4
- gpt-3.5-turbo

**Claude:**
- claude-sonnet-4-20250514 (Extended Thinking)
- claude-3-5-sonnet-20241022 (с vision)
- claude-3-opus-20240229 (с vision)
- claude-3-sonnet-20240229 (с vision)
- claude-3-haiku-20240307

### 📊 Поддерживаемые форматы файлов

**Текстовые (30+):**
- Код: .py, .js, .ts, .java, .cpp, .go, .rs, и др.
- Данные: .json, .xml, .yaml, .csv
- Документы: .txt, .md, .html, .css

**Изображения:**
- .jpg, .jpeg, .png, .gif, .bmp, .webp

### 🎯 Примеры использования

```python
from chat_bot import ChatBot

# Создание с выбором модели
bot = ChatBot(model="gpt-4o")

# Смена модели
bot.set_model("claude-3-5-sonnet-20241022")

# Анализ текстового файла
response = bot.chat("Объясни этот код", file_path="script.py")

# Анализ изображения (для моделей с vision)
response = bot.chat("Что на картинке?", file_path="image.jpg")
```

## Версия 2.2 - Унификация через ProxyAPI (2026-03-01)

### ✨ Ключевые изменения
- **Единый API ключ ProxyAPI** для OpenAI и Claude
- **Упрощенная конфигурация** - только один ключ вместо двух
- **Claude через ProxyAPI** - `https://api.proxyapi.ru/anthropic`
- **Удалена зависимость** от прямого ключа Anthropic

### 🔧 Технические изменения

#### testagent.py
- Изменен `base_url` на `https://api.proxyapi.ru/anthropic`
- Используется `PROXYAPI_KEY` вместо `ANTHROPIC_API_KEY`
- Добавлен параметр `api_key` в функцию `chat_with_thinking_model()`

#### chat_bot.py
- Удален параметр `anthropic_key` из конструктора
- Используется единый `api_key` для обоих режимов
- Передача `api_key` в `chat_with_thinking_model()`

#### test_chat.py
- Удалена переменная `ANTHROPIC_API_KEY`
- Используется только `API_KEY` (ProxyAPI)

#### Конфигурация
- `.env.example` - удален `ANTHROPIC_API_KEY`
- Только `PROXYAPI_KEY` требуется

### 📚 Документация
- Обновлены все документы с информацией о едином ключе
- Упрощены инструкции по настройке
- Обновлены примеры конфигурации

### ✅ Преимущества
- ✅ Один ключ вместо двух
- ✅ Упрощенная настройка
- ✅ Единая точка доступа
- ✅ Меньше конфигурации
- ✅ Проще для новых пользователей

## Версия 2.1 - Интеграция Claude Extended Thinking (2026-03-01)

### ✨ Новые возможности
- **Интеграция Claude Extended Thinking API** от Anthropic
- **Два режима работы**: Thinking (Claude) и Normal (OpenAI)
- **Процесс размышления модели** - видите как модель думает
- **Настройка бюджета токенов** для размышлений (default=1500)
- **Автоматическое переключение режимов** в интерактивном чате
- **Красивое форматирование вывода** с разделами для мышления и ответа

### 📝 Новые файлы
- `testagent.py` - модуль для работы с Claude Extended Thinking
- `CLAUDE_INTEGRATION.md` - полное руководство по интеграции
- `EXAMPLES_CLAUDE.md` - примеры использования думающей модели
- `API_REFERENCE.md` - справочник по API

### 🔧 Изменения в API

#### chat_bot.py
- Добавлены параметры: `mode`, `claude_model`, `thinking_budget`, `anthropic_key`
- Метод `chat()` теперь возвращает `dict` для thinking mode
- Новые методы: `set_mode()`, `get_mode()`

#### test_chat.py
- Добавлен выбор режима при запуске
- Функция `generate_response()` поддерживает параметр `mode`
- Команда `режим` для переключения в интерактивном чате
- Импорт функций из `testagent.py`

### 📦 Зависимости
- Добавлена `anthropic>=0.40.0` для работы с Claude API

### 🎯 Конфигурация
- Новая переменная окружения: `ANTHROPIC_API_KEY`
- Новые параметры: `CLAUDE_MODEL`, `THINKING_BUDGET`, `CURRENT_MODE`

### 📚 Документация
- Обновлен `README.md` с информацией о двух режимах
- Добавлено 3 новых документа по Claude
- Обновлены примеры использования
