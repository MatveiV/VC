**Требование:** Добавить возможность выбора нескольких других моделей ИИ с указанием их версий.

  - Провайдер (OpenAI/Claude)

  - Максимальный контекст
  - Поддержка thinking/vision
  - Стоимость (вход/выход)

#### Функциональность выбора

### 2. Работа с файлами ✅

**Требование:** Добавить возможность загружать файл для анализа и преобразования его моделью ИИ.

**Реализация:**

#### Создан модуль `file_handler.py`
- **30+ текстовых форматов**: py, js, ts, java, cpp, json, xml, md, и др.
- **6 форматов изображений**: jpg, png, gif, bmp, webp
- **Автоматическое определение** типа файла
- **Поддержка кодировок**: utf-8, cp1251, latin-1, utf-16

#### Функциональность работы с файлами
```python
# Анализ текстового файла
response = bot.chat("Объясни этот код", file_path="script.py")

# Анализ изображения (для моделей с vision)
response = bot.chat("Что на картинке?", file_path="photo.jpg")

# Загрузка файла вручную
from file_handler import load_file
file_info = load_file("document.txt")
```

## 🏗️ Архитектура

### Модульная структура

```
chat_bot.py (основной модуль)
    ↓
    ├── models_config.py (конфигурация моделей)
    │   ├── ModelInfo (dataclass)
    │   ├── OPENAI_MODELS (4 модели)
    │   ├── CLAUDE_MODELS (5 моделей)
    │   └── Функции работы с моделями
    │
    └── file_handler.py (работа с файлами)
        ├── FileInfo (dataclass)
        ├── load_file() (загрузка)
        ├── format_file_for_prompt() (текст)
        └── create_vision_message() (изображения)
```

### Поток работы с файлами

```
1. Пользователь → bot.chat(message, file_path="file.py")
2. ChatBot → load_file(file_path)
3. file_handler → определение типа файла
4. file_handler → загрузка содержимого
5. ChatBot → форматирование для AI
6. API → обработка запроса
7. ChatBot → возврат ответа
```

### Поток выбора модели

```
1. Пользователь → bot.set_model("gpt-4o")
2. ChatBot → get_model_info("gpt-4o")
3. models_config → возврат ModelInfo
4. ChatBot → настройка параметров
5. ChatBot → автоопределение режима (thinking/normal)
```

## 🎓 Best Practices

### 1. Выбор модели

```python
# Для анализа изображений

# Для баланса цена/качество

```

### 2. Работа с файлами

```python
# Проверка поддержки формата
from file_handler import get_supported_formats
formats = get_supported_formats()

# Обработка ошибок
try:
    response = bot.chat("Анализ", file_path="file.xyz")
except ValueError as e:
    print(f"Неподдерживаемый формат: {e}")
except FileNotFoundError as e:
    print(f"Файл не найден: {e}")
```
