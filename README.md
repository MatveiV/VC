# Мини-кошелёк для путешественника

Telegram-бот для управления расходами во время путешествий в разных валютах. Позволяет отслеживать расходы, планировать бюджет и получать актуальные курсы валют.

## Возможности

- Создание нескольких путешествий с разными валютами
- Отслеживание расходов в иностранной валюте
- Конвертация сумм по актуальному курсу
- Категории расходов (транспорт, жильё, еда и др.)
- Планирование бюджета по категориям
- Предупреждения о превышении лимитов
- Визуализация расходов в виде графиков
- Хранение данных в SQLite

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd wallet-bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` в корневой директории и добавьте туда следующие переменные:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
EXCHANGE_RATE_API_KEY=ваш_api_ключ_от_exchangerate_host
```

4. Запустите бота:
```bash
python bot.py
```

## Использование

### Команды бота

- `/start` - начать работу с ботом
- `/newtrip` - создать новое путешествие
- `/switch` - переключиться между путешествиями
- `/balance` - посмотреть баланс
- `/history` - история расходов
- `/setrate` - изменить курс валюты

### Основной сценарий

1. **Создание путешествия**:
   - Бот запросит название путешествия
   - Код исходной валюты (например, RUB)
   - Код целевой валюты (например, USD)
   - Начальную сумму

2. **Добавление расходов**:
   - Просто отправьте число (например, "100") - это будет воспринято как расход
   - Бот покажет конвертацию и попросит подтвердить
   - Выберите категорию расхода

3. **Просмотр информации**:
   - Используйте команды или кнопки меню для просмотра баланса и истории

## Структура базы данных

### Таблица `users`
- `user_id` - ID пользователя в Telegram
- `username` - имя пользователя
- `created_at` - дата создания

### Таблица `trips`
- `trip_id` - ID путешествия
- `user_id` - ID владельца
- `name` - название путешествия
- `from_currency` - исходная валюта
- `to_currency` - целевая валюта
- `rate` - курс обмена
- `balance_from` - баланс в исходной валюте
- `balance_to` - баланс в целевой валюте
- `created_at` - дата создания
- `is_active` - активно ли путешествие

### Таблица `expenses`
- `expense_id` - ID расхода
- `trip_id` - ID путешествия
- `amount` - сумма расхода
- `currency` - валюта расхода
- `category` - категория расхода
- `note` - заметка
- `timestamp` - дата и время
- `rate_at_time` - курс на момент расхода

### Таблица `budgets`
- `budget_id` - ID бюджета
- `trip_id` - ID путешествия
- `category` - категория расхода
- `limit_amount` - лимит
- `spent_amount` - потрачено

## API

Проект использует API от [exchangerate.host](https://exchangerate.host/) для получения курсов валют.

## Файлы проекта

- `bot.py` - основной код бота
- `database.py` - работа с базой данных
- `api_client.py` - клиент для работы с API
- `config.py` - конфигурация
- `requirements.txt` - зависимости
