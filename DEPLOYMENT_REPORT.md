### 4. Обработка ошибок и таймаутов ✅

**Требование:** Обработка ошибок запросов и таймаутов

**Реализация:**

#### Таймауты ✅
```python
bot = ChatBot(
    timeout=60,        # Таймаут запросов (секунды)
    max_retries=3      # Количество повторных попыток
)
```

#### Обработка ошибок ✅
- ✅ Try-catch блоки для всех API запросов
- ✅ Автоматические повторные попытки с экспоненциальной задержкой
- ✅ Логирование всех ошибок
- ✅ Понятные сообщения об ошибках пользователю
- ✅ Graceful degradation (переключение с thinking на normal при ошибке)

**Пример обработки:**
```python
for attempt in range(self.max_retries):
    try:
        response = self.client.chat.completions.create(...)
        return response
    except Exception as e:
        logger.error(f"Ошибка (попытка {attempt + 1}): {e}")
        if attempt < self.max_retries - 1:
            wait_time = 2 ** attempt
            time.sleep(wait_time)
        else:
            raise Exception(f"Не удалось после {self.max_retries} попыток")
```
