# Возможности проекта

## Основные функции

### 1. Работа через ProxyAPI.ru
- Все запросы идут через `https://api.proxyapi.ru/openai/v1`
- Используется API ключ от ProxyAPI
- Полная совместимость с OpenAI SDK

### 2. Сохранение контекста диалога
- История автоматически сохраняется в JSON файлы после каждого сообщения
- При следующем запуске история загружается автоматически
- Бот помнит предыдущие разговоры между запусками программы

### 3. Поддержка системных промптов
- Настройка поведения бота через системные сообщения
- Системный промпт сохраняется вместе с историей

### 4. Управление историей
- Команда `сброс` для очистки истории
- Возможность указать свой файл для истории
- Автоматическое создание и удаление файлов истории

## Файлы проекта

### Основные модули
- `chat_bot.py` - Класс ChatBot с полным функционалом
- `test_chat.py` - Функции для работы с API и интерактивный режим

### Тестовые файлы
- `quick_test.py` - Быстрая проверка работоспособности
- `test_persistence.py` - Тест сохранения контекста между запусками
- `test_save_load.py` - Простой тест сохранения/загрузки
- `demo_persistence.py` - Интерактивная демонстрация сохранения контекста

### Конфигурация
- `.env` - Переменные окружения (API ключ)
- `.env.example` - Шаблон для .env
- `requirements.txt` - Зависимости проекта
- `.gitignore` - Исключения для git

## Примеры использования

### Пример 1: Простой диалог
```python
from chat_bot import ChatBot

bot = ChatBot()
bot.set_system_prompt("Ты помощник программиста.")
response = bot.chat("Как работает async/await в Python?")
print(response)
```

### Пример 2: Диалог с сохранением
```python
from chat_bot import ChatBot

# Первый запуск
bot = ChatBot(history_file="my_chat.json")
bot.chat("Меня зовут Иван")

# Второй запуск (в другой раз)
bot = ChatBot(history_file="my_chat.json")
response = bot.chat("Как меня зовут?")
# Ответ: "Тебя зовут Иван"
```

### Пример 3: Очистка истории
```python
from chat_bot import ChatBot

bot = ChatBot()
bot.chat("Привет!")
bot.reset_conversation()  # Очищает историю и удаляет файл
```

## Технические детали

### Зависимости
- `openai>=1.0.0` - SDK для работы с OpenAI API
- `python-dotenv>=1.0.0` - Загрузка переменных окружения

### Поддерживаемые модели
- gpt-3.5-turbo (по умолчанию)
- gpt-4
- gpt-4o
- Любые другие модели, доступные через ProxyAPI

### Обработка ошибок
- Проверка наличия API ключа при инициализации
- Обработка ошибок кодировки при сохранении
- Graceful fallback при ошибках загрузки истории

## Рекомендации

1. **Безопасность**: Не коммитьте файл `.env` с реальным API ключом
2. **История**: Файлы истории автоматически добавлены в `.gitignore`
3. **Тестирование**: Используйте `test_persistence.py` для проверки сохранения контекста
4. **Демонстрация**: Запустите `demo_persistence.py` несколько раз для наглядной демонстрации
